# Python Pentools

A collection of simple penetration testing tools made using Python. These tools were made to complete the [freeCodeCamp Information Security course](https://www.freecodecamp.org/learn/information-security).

## Getting Started

### Installation

Clone the repository:

```bash
git clone https://github.com/ElliottStorey/python-pentools.git
cd python-pentools
```

### Usage

Run any of the tools with Python:

```bash
python PortScanner.py
python NmapScanner.py
python BannerGrabber.py
python TCPClient.py
python TCPServer.py
```
